# Running Man Animation

A Go program that displays an animated running man in the terminal.

<img width="708" height="594" alt="runningman" src="https://github.com/user-attachments/assets/eee0cd63-f8f8-4389-b947-df372028b93d" />

## Build

```bash
go build -o runningman .
```

## Run

```bash
./runningman
```

Press `Ctrl+C` to stop the animation.

## Requirements

- Go 1.16 or later
